# Plataforma OE — versão leve para GitHub Pages

Esta versão foi otimizada para publicação pelo site do GitHub.

## Publicação

1. Extraia o ZIP no computador.
2. Abra o repositório `boasvindas` no GitHub.
3. Entre na pasta usada pelo GitHub Pages.
4. Clique em **Add file > Upload files**.
5. Envie **os arquivos e pastas que estão dentro desta pasta**, não o ZIP.
6. Confirme o commit.

## Importante

- As 32 artes foram convertidas de PNG para WebP, mantendo 941 × 1672 px.
- O pacote completo ficou abaixo do limite comum de upload do GitHub pelo navegador.
- Não renomeie nem mova a pasta `assets/gibi`.
- Antes de publicar, mantenha no `config.js` a URL do Google Apps Script já usada no site atual para a confirmação de leitura.
